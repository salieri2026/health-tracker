初始設定
⚡ 首次使用請先完成以下設定
1. 前往 Google Sheets 建立一個新試算表
2. 點選上方選單 「擴充功能」→「Apps Script」
3. 刪除預設程式碼，貼上 Code.gs 的內容，儲存
4. 點選 「部署」→「新增部署作業」，類型選 Web 應用程式
5. 執行身分選 我，存取權選 任何人，點選部署
6. 複製取得的 Web App 網址，貼到下方
7. 你的 Claude API Key 貼到第二格（用於 AI 功能）

API 連線設定
詳細設定步驟
  1. 開啟 sheets.google.com，登入 Google 帳號
  2. 點「空白試算表」建立新檔案，命名為「健康追蹤」
  3. 上方選單：擴充功能 → Apps Script
  4. 在 Apps Script 編輯器中，刪除 function myFunction(){}
  5. 開啟下載的 Code.gs，複製全部內容貼入
  6. Ctrl+S 儲存，右上角「部署」→「新增部署作業」
  7. 類型：Web 應用程式 / 描述：health-tracker
  8. 執行身分：我 / 誰可以存取：任何人
  9. 按「部署」，複製 Web 應用程式網址
  10. 回到這個設定頁面，貼上網址並儲存
